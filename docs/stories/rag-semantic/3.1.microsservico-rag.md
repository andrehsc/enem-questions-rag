# Story 3.1: Microsserviço RAG

## Status
Draft

## Story
**As a** sistema TeachersHub,
**I want** acessar capacidades de análise semântica via microsserviços Python,
**so that** professores recebam sugestões inteligentes de questões com base em similaridade semântica.

## Acceptance Criteria
1. Microsserviço Python FastAPI operacional com endpoints RAG em `python-ml-services/rag-service/`
2. Integração com modelos de embedding (sentence-transformers) para análise semântica das questões
3. Cache Redis implementado para consultas frequentes com TTL configurável
4. APIs documentadas via Swagger/OpenAPI (`/docs` e `/openapi.json`)
5. Performance atende NFR3: resposta < 3 segundos para consultas RAG simples
6. Health check endpoint (`/health`) retorna status dos componentes (DB, Redis, modelo de embedding)

### Arquitetura RAG
[Source: docs/architecture.md#python-ml-microsservicos]
- TeachersHub .NET chama `POST /api/v1/semantic-search` com query e filtros
- RAG service verifica cache Redis antes de chamar o modelo
- Busca vetorial no PostgreSQL (pgvector) com operador `<=>` (cosine similarity)
- Resultados incluem score de similaridade para ranking

### Schema pgvector
```sql
-- Extensão já habilitada no Epic 1/2
-- CREATE EXTENSION IF NOT EXISTS vector;

CREATE TABLE question_embeddings (
    id SERIAL PRIMARY KEY,
    question_id INTEGER NOT NULL REFERENCES questions(id) ON DELETE CASCADE,
    embedding vector(384) NOT NULL,
    model_version VARCHAR(100) DEFAULT 'paraphrase-multilingual-MiniLM-L12-v2',
    created_at TIMESTAMP DEFAULT NOW()
);

-- Índice HNSW para busca aproximada eficiente
CREATE INDEX idx_question_embeddings_hnsw
    ON question_embeddings USING hnsw (embedding vector_cosine_ops)
    WITH (m = 16, ef_construction = 64);
```

**Busca por similaridade:**
```sql
SELECT q.id, q.question_text, q.subject,
       1 - (qe.embedding <=> $1::vector) AS similarity_score
FROM question_embeddings qe
JOIN questions q ON q.id = qe.question_id
WHERE q.subject = $2
ORDER BY qe.embedding <=> $1::vector
LIMIT $3;
```

### Modelo de Embedding
- Usar `paraphrase-multilingual-MiniLM-L12-v2` para suporte nativo ao português
- Dimensão de embedding: 384
- Mountar volume Docker para cache do modelo em `/root/.cache/sentence_transformers`

```yaml
    - redis
  networks:
    - teachershub-network
```
