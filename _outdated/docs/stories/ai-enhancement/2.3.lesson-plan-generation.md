# Story 2.3: AI-Powered Lesson Plan Generation

## Status
Draft

## Story

Como professor utilizando a plataforma ENEM RAG,
Eu quero gerar planos de aula automaticamente baseados em questões ENEM selecionadas e objetivos pedagógicos,
Para que eu possa criar aulas estruturadas e personalizadas em minutos ao invés de horas de planejamento manual.

## Context Source

- **Documento Base:** docs/architecture.md - "Fluxo 2: Geração de Planos de Aula"
- **Enhancement Type:** Advanced AI Feature - Fase 3 (4 semanas)  
- **Existing System Impact:** Nova funcionalidade integrada ao TeachersHub existente

## Acceptance Criteria

**Functional Requirements:**

1. **Question-Based Plan Generation** - IA gera planos de aula baseados em questões ENEM selecionadas
2. **Customizable Parameters** - Duração (30-120min), público-alvo, objetivos de aprendizagem
3. **Structured Output** - Planos com objetivos, metodologia, cronograma, recursos, avaliação
4. **Multi-Subject Support** - Funciona para todas as matérias ENEM (Linguagens, Humanas, Natureza, Matemática)

**Integration Requirements:**

5. **TeachersHub Integration** - Nova funcionalidade acessível via ASP.NET Core backend
6. **Question Selection UI** - Interface para escolher questões do sistema RAG existente
7. **Plan Storage** - Planos salvos no banco de dados com versionamento
8. **Export Capabilities** - Planos exportáveis em PDF, Word, ou markdown

**Quality Requirements:**

9. **Generation Speed** - Plano completo gerado em < 30s para até 10 questões
10. **Educational Quality** - Planos seguem metodologias pedagógicas reconhecidas
11. **Customization Flexibility** - Professores podem editar todos os aspectos dos planos
12. **Template Consistency** - Outputs seguem padrão estrutural consistente

### Database Schema

```sql
-- TeachersHub lesson plans table
CREATE TABLE teachershub.lesson_plans (
    id SERIAL PRIMARY KEY,
    title VARCHAR(200) NOT NULL,
    user_id VARCHAR(450) REFERENCES AspNetUsers(Id), -- ASP.NET Identity
    duration_minutes INTEGER NOT NULL,
    target_audience VARCHAR(100),
    learning_objectives TEXT,
    content JSONB NOT NULL, -- Structured plan content
    selected_question_ids INTEGER[],
    primary_subject VARCHAR(50),
    methodology VARCHAR(50),
    generated_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW(),
    is_public BOOLEAN DEFAULT FALSE,
    share_code VARCHAR(20) UNIQUE, -- For sharing plans
    
    -- Constraints
    CONSTRAINT duration_valid CHECK (duration_minutes >= 30 AND duration_minutes <= 180),
    CONSTRAINT questions_limit CHECK (array_length(selected_question_ids, 1) <= 15)
);

-- Indices for performance
CREATE INDEX idx_lesson_plans_user_id ON teachershub.lesson_plans(user_id);
CREATE INDEX idx_lesson_plans_subject ON teachershub.lesson_plans(primary_subject);
CREATE INDEX idx_lesson_plans_created ON teachershub.lesson_plans(generated_at);
CREATE INDEX idx_lesson_plans_share_code ON teachershub.lesson_plans(share_code);

-- Full text search on content
CREATE INDEX idx_lesson_plans_content_search ON teachershub.lesson_plans 
USING gin(to_tsvector('portuguese', title || ' ' || learning_objectives));
```

### Performance Considerations

```csharp
// Async generation with progress tracking
public async Task<LessonPlan> GenerateLessonPlanAsync(
    LessonPlanRequest request, 
    IProgress<GenerationProgress> progress = null)
{
    progress?.Report(new GenerationProgress("Analyzing selected questions...", 10));
    var questions = await FetchQuestionsWithAnalysis(request.QuestionIds);
    
    progress?.Report(new GenerationProgress("Building educational context...", 30));
    var context = await BuildEducationalContext(questions, request);
    
    progress?.Report(new GenerationProgress("Generating lesson plan...", 60));
    var plan = await _aiService.GeneratePlan(context);
    
    progress?.Report(new GenerationProgress("Structuring and validating...", 90));
    var structured = await StructureAndValidate(plan);
    
    progress?.Report(new GenerationProgress("Saving to database...", 100));
    return await _repository.SaveAsync(structured);
}
```
